========= 사용 방법  =========
1. Java 7 설치하기
네이버 크롤러는 Java 7으로 만들어졌습니다.
그래서 Java 7이 있어야 될어가니까 Java가 깔려있지 않다면
Oracle 홈페이지에 가서 설치하세요. 홈페이지 주소는 
http://www.oracle.com/technetwork/java/javase/downloads/jre7-downloads-1880261.html

2. 실행
[Windows]
WebtoonCrawler.jar 을 더블클릭하세요.

[Mac/Linux]
콘솔을 열고, WebtoonCrawler.jar가 저장된 곳으로 
이동한 다음 "java -jar WebtoonCrawler.jar"을 입력하세요.

예를 들어 WebtoonCrawler.jar가 '/Users/Seulgi/funnyDir' 
디렉토리에 저장되어 있으면 아래 두 명령어를 순서대로 입력하면 
실행이 되요.
첫번째: cd /Users/Seulgi/funnyDir
두번째: java -jar WebtoonCrawler.jar
